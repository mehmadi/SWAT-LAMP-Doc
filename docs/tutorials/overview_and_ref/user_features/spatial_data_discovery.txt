======================
Spatial Data Discovery
======================

SWAT-LAMP allows users to browse and search for geospatial data. 
By combining collaboration found in social networks with specialized geospatial tools, SWAT-LAMP makes it easy to explore, process, style, and share maps and geospatial data.
Spatial datasets can be imported and shared, all through a non-technical user interface.

Features include:

* Powerful spatial search engine

* Federated OGC services

* Metadata catalogue
